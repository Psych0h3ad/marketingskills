# Marketing Skills for Manus

This is a fork of [syntax-syndicate/marketing-skills](https://github.com/syntax-syndicate/marketing-skills) adapted for use with Manus AI.

## Installation for Manus

### Method 1: Import from GitHub (Recommended)

1. Open Manus Skills settings
2. Click "Add custom Skills"
3. Select "Import from GitHub"
4. Enter the URL: `https://github.com/Psych0h3ad/marketingskills`
5. Click "Import"

### Method 2: Import Individual Skills

Each skill can be imported individually by navigating to the skill directory URL:

```
https://github.com/Psych0h3ad/marketingskills/tree/main/skills/[skill-name]
```
